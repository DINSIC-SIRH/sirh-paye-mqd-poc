STIFSCRUM-1616 : Interdire à un administrateur de s'auto supprimer ou de s'enlever le profil administrateur

1: Se connecter à l'IHM avec un profil administrateur
2: Sélectionner "Gestion des Comptes" dans la barre de menu
3: Dans la zone de saisie d'un utilisateur taper le nom et prénom de l'administrateur de connection
4: Vérifier que la check box "Administrateur" est cochée et non modifiable
5: Vérifier que le bouton "Supprimer" est grisé.
6: Dans la zone de saisie d'un utilisateur taper le nom et prénom d'un autre administrateur
7: Vérifier que la check box "Administrateur" est cochée mais modifiable
8: Vérifier que le bouton "Supprimer" n'est pas grisé.
9: Dans la zone de saisie d'un utilisateur taper le nom et prénom d'un autre utilisateur n'ayant pas le profil administrateur
10: Vérifier que la check box "Administrateur" n'est pas cochée mais modifiable
11: Vérifier que le bouton "Supprimer" n'est pas grisé.
